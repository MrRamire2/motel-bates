Tecnología Utilizada en el Proyecto

Hardware

  • Computador personal: Estación de trabajo principal para el desarrollo.
  • Servidor web: Configuración basada en Linux con Apache y MariaDB.

Software

  • PHP: Lenguaje principal del lado del servidor.
  • HTML: Marcado para la estructura de las páginas web.
  • JavaScript: Interactividad y lógica en el cliente.
  • CSS: Estilización y diseño de la interfaz.
  • Base de Datos:
    • MariaDB: Para la gestión de las bases de datos principales.
    • SQL: Lenguaje utilizado para la creación y gestión de las tablas y procedimientos almacenados.
  • Control de Versiones: Git y GitHub para el seguimiento del código y la colaboración.
  • Entorno de Desarrollo: Visual Studio Code utilizado como IDE principal.
